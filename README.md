## Mandelbrot Trajectory Infima

These renders below are the result of considering the "trajectories" of points inside the mandelbrot
set when iterating <img alt="z \mapsto z^2 + c" src="https://render.githubusercontent.com/render/math?math=z%20%5Cmapsto%20z%5E2%20%2B%20c" style="transform: translateY(20%);" />. Specifically, looking at the closest point on such a
trajectory to the origin <img alt="a(c) = \inf \{\left|z_n\right|\} = \inf \{\left|P_c^n(0)\right|\}" src="https://render.githubusercontent.com/render/math?math=a%28c%29%20%3D%20%5Cinf%20%5C%7B%5Cleft%7Cz_n%5Cright%7C%5C%7D%20%3D%20%5Cinf%20%5C%7B%5Cleft%7CP_c%5En%280%29%5Cright%7C%5C%7D" style="transform: translateY(20%);" />. As far
as I can tell, this was first described / explored in a 1986 book "The Beauty of Fractals".
